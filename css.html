CSS

Introduction

L'anatomie de base de la syntaxe CSS écrite pour les styles en ligne et les feuilles de style.
Certains termes CSS couramment utilisés, tels que règles, sélecteur et déclaration.
Les styles en ligne CSS peuvent être écrits à l'intérieur de la balise HTML d'ouverture en utilisant l'attribut style.
Les styles en ligne peuvent être utilisés pour styliser HTML, mais ce n'est pas la meilleure pratique.
Une feuille de style interne est écrite en utilisant l'élément "style" à l'intérieur de l'élément "head" d'un fichier HTML.
Les feuilles de style internes peuvent être utilisées pour styliser HTML mais ne sont également pas la meilleure pratique.
Une feuille de style externe sépare le code CSS du HTML, en utilisant l'extension de fichier .css.
Les feuilles de style externes sont la meilleure approche en ce qui concerne l'utilisation de HTML et CSS.
Les feuilles de style externes sont liées à HTML en utilisant l'élément .

Les Selecteurs

CSS peut sélectionner des éléments HTML par type, classe, ID et attribut.
Tous les éléments peuvent être sélectionnés en utilisant le sélecteur universel.
Un élément peut avoir différents états en utilisant le sélecteur de pseudo-classe.
Plusieurs classes CSS peuvent être appliquées à un seul élément HTML.
Les classes peuvent être réutilisables, tandis que les ID ne peuvent être utilisés qu'une seule fois.
Les ID sont plus spécifiques que les classes, et les classes sont plus spécifiques que le type. Cela signifie que les ID remplaceront tous les styles d'une classe, et les classes remplaceront tous les styles d'un sélecteur de type.
Plusieurs sélecteurs peuvent être enchaînés pour sélectionner un élément. Cela augmente la spécificité mais peut être nécessaire.
Les éléments imbriqués peuvent être sélectionnés en séparant les sélecteurs par un espace.
Plusieurs sélecteurs non liés peuvent recevoir les mêmes styles en séparant les noms des sélecteurs par des virgules.

Les regles visuel

When setting typefaces on a web page, keep the following points in mind:
The font specified must be installed on the user’s computer or downloaded with the site.
Web safe fonts are a group of fonts supported across most browsers and operating systems.
Unless you are using web safe fonts, the font you choose may not appear the same between all browsers and operating systems.
When the name of a typeface consists of more than one word, it’s a best practice to enclose the typeface’s name in quotes, like so: 

  h1 {
            font-family: 'Courier New'}
